LOGISTIC REGRESSION

The following libraries need to be installed : 
1) numpy
2) matplotlib

RUNNING INSTRUCTIONS : 

1) Place all the files in a desired location

2) In the python shell, excecute the .py file
	
	>>python logreg.py
	
3) Output for 3-a will be printed on the console (gradients) and will also be generated as 4 separate graphs for each of the learning rates.
   Output for 3-b will appear as a graph (takes a while to run) which contains plots for training and test accuracies against the different penalty terms
